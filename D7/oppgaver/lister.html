Definisjonsliste

10 uttrykk vi har lært i utvikling til nå:
HTML
HyperTextMarkupLanguage - benyttes til å lage websider
CSS
Cascading Styling Sheet - benyttes til å style HTML elementer
Javasccript
Programmeringsspråket som benyttes for å gjøre websider interaktive
IDE
Integrated Development Environment - Et utvikler verktøy som VS Code
VS Code
Visual Studio Code - En gratis IDE fra Microsoft som kan benyttes til å lage websider
Discord
Discord er en kommunikasjonsplattform for communities hvor man kan chatte, dele filer osv.
Favicon
Et favicon er et ikon som benyttes av websider for å gi dem et spesifikt ikon i tab og i favoritter
index.html
Filen som har innholdet til startsiden til nettsteder
Front-End
Den delen av webutvikling som omhandles av HTML, CSS og Javascript og kjøres lokalt hos brukeren i nettleser
Back-End
Den delen av nettsider som kjøres sentralt en tjeneste som webserver, databaseserver
API
Application Programmable Interface - Et grensesnitt til en ekstern tjeneste for å etterspørre f.eks værvarsel for Bergen fra yr.no
